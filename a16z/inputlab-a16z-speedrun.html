How it works

Take the text from the app. Fix it. Put it back.

InputLab uses the Cmd+C and Cmd+V path, but the user stays in the same window.

1

Use selected text, clipboard text, or a voice recording.

2

3

Transform

Rewrite, shorten, translate, correct, or turn speech into polished text.

4

Return

Copy or auto-paste the result into the active app.

Problem

People already use AI to edit text. The problem is the round trip.

Copy text, open ChatGPT, explain the task, paste context, copy the answer, return to the app, fix formatting.

Why now

Voice becomes useful when AI cleans it up.

Normal dictation gives a transcript. InputLab turns the transcript into the message, update, reply, or note you meant.

Insight

The best AI editor should work where the cursor already is.

No new editor to adopt.
No prompt to rewrite every time.
No context switch before the text is usable.
